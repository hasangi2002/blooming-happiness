# 🌸 Blooming Happiness

Luxury flower boutique — full-stack e-commerce platform.

## Tech Stack

**Frontend:** React 19, Vite, TailwindCSS, React Router, Framer Motion, GSAP, React Hook Form, Swiper, React Icons, Axios, Lottie

**Backend:** Node.js, Express.js, MongoDB Atlas (Mongoose), JWT Auth, Cloudinary, Multer

**Payments:** Stripe, PayHere

**Maps:** Google Maps API

## Project Structure

\```
blooming-happiness/
├── frontend/    # React + Vite client
├── backend/     # Express API
└── docs/        # Project documentation
\```

## Getting Started

\```bash
npm install
npm run install:all

cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env

npm run dev
\```

Frontend: http://localhost:5173
Backend:  http://localhost:5000
